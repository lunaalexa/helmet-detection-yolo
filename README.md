# Helmet Detection for Construction Safety using YOLO

Deep Learning project exploring YOLO-based object detection for construction site safety monitoring.

## Technologies
- Python
- YOLOv8
- YOLOv11
- Ultralytics

## Key Result
- 92.3% Precision
- 91.6% mAP@50
